🤖 Bot Pelacak Pengeluaran Indonesia

Server webhook untuk bot Telegram pelacak pengeluaran

✅ Server sedang berjalan dan siap menerima webhook

📸 Analisis Foto Struk

Bot dapat membaca dan menganalisis foto struk pembelian menggunakan AI untuk mencatat pengeluaran secara otomatis.

🎤 Voice Note Recognition

Kirim voice note ke bot untuk mencatat pengeluaran dengan suara. Bot akan memproses audio dan mengekstrak informasi pengeluaran.

📊 Integrasi Google Sheets

Semua data pengeluaran dan pemasukan disimpan otomatis ke Google Sheets untuk akses mudah dan analisis lanjutan.

📈 Rekap Fleksibel

Dapatkan ringkasan pengeluaran harian, bulanan, tahunan, atau periode custom sesuai kebutuhan.

🔧 Perintah Bot Tersedia:

/start - Memulai bot dan melihat panduan /help - Bantuan lengkap penggunaan bot /pengeluaran [jumlah] [kategori] [keterangan] /pemasukan [jumlah] [kategori] [keterangan] /rekapharian [tanggal] - Rekap harian /rekapcustom [rentang tanggal] - Rekap custom /rekapbulanan [bulan tahun] - Rekap bulanan /rekaptahunan [tahun] - Rekap tahunan

🌏 Dukungan Bahasa Indonesia

Bot dirancang khusus untuk pengguna Indonesia dengan dukungan format tanggal, mata uang, dan bahasa Indonesia.
